My
Work

Vapour is my newest creation, in collaboration with Ajiya7, mariea1022, and crzn24.

This website is an unofficial extension of the Steam Client. Creating a new account automatically saves all of their games and achievements, given they provide a working Steam ID. The website has features that pick a random game for them to play, as well as going for a random achievement they don't already have. It also has a feedback system where users can leave comments about their experiences obtaining achievements for their Steam games.

Weather Dashboard

This website utilizes Open Weather Map API to allow users to search for weather in thousands of cities. All searches are saved in the "Previous Searches" section. This gives users the ability to search their favorite cities at the touch of a button!

Work Day Scheduler

Struggle with organization at work? This is the website for you! Each work block (defined as 1 hour) is color coordinated based upon if it is in the past, present, or future. Inside each time block, users can enter and save events and tasks for each time block. This makes it easy to know when you are ahead or behind on your daily work tasks. It also allows for flexibility as they can be edited and saved at any time.

Code Quiz

Want to learn JavaScript? This website will help you learn some terminology by taking a quiz! How fast you complete your quiz, as well as your accuracy, will determine your score. Highscores can be saved on each completion and may be accessed at any time in the upper-left corner of your screen. Happy coding!

Password Generator

Do you use the same password for every website? You know you shouldn't... but you probably still do it. Now, it's easy to STOP doing that with Password Generator! With the click of a button, you can generate a random and secure password based upon parameters that you will set.
